# Event Management System

## Project Overview
Welcome to the **Event Management System**, a streamlined platform for organizing and managing events efficiently. This system is designed to cater to both **users** looking to book events and **administrators** managing event details, registrations, and user engagement.

---
## Key Features
- **User Authentication** ‚Äì Secure login & signup process
- **Event Browsing** ‚Äì Search and explore various events
- **Ticket Booking** ‚Äì Seamless reservation system
- **Admin Dashboard** ‚Äì Manage users, bookings, and events
- **Optimized Performance** ‚Äì Fast and scalable infrastructure

---
## Tech Stack
- **Frontend:** React.js, HTML, CSS
- **Backend:** Node.js, Express.js, MongoDB
- **Authentication:** JSON Web Tokens (JWT)
- **API Testing:** Postman

## Getting Started

### Setting Up the Frontend
1. Open **VS Code** and navigate to the project folder.
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
4. The application will launch in **Google Chrome** automatically.

### Setting Up the Admin Panel
1. Open the **VS Code terminal**.
2. Run:
   ```sh
   npm install
   ```
3. Start the admin dashboard:
   ```sh
   npm start
   ```
4. Access the admin interface in your browser.

### Running the Backend
1. Install **Node.js** from [here](https://nodejs.org/en/download).
2. Navigate to the backend folder.
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the server:
   ```sh
   node index.js
   ```

## How to Use the Application

### **For Users:**
1. **Sign up/Login** using email and password.
2. **Browse Events** ‚Äì Search for upcoming events.
3. **Book Tickets** ‚Äì Secure your spot and get a confirmation.
4. **Manage Bookings** ‚Äì View and manage your reservations.

### **For Admins:**
1. **Login as Admin.**
2. **Manage Events** ‚Äì Add, update, or remove events.
3. **User Management** ‚Äì Track and handle user activities.
4. **Handle Bookings** ‚Äì Approve or modify event reservations.
